# Task Creation Screen

Allows users to create new tasks, potentially using voice input or text.

## UI Structure

### Header
- **Title**: "Создать задачи" (Create Tasks)
- **Back Button**: Returns to Main Menu.

### Input Methods
- **Voice Input Button**:
  - Icon: Microphone
  - Label: "Голосовой ввод" (implied)
  - Action: Starts voice recognition to create a task.

- **Text Input**:
  - Placeholder: "Введите задачу..."
  - Action: Allows manual typing.

### Task List
- **Section Title**: "Активные задачи" (Active Tasks)
- **List Items**:
  - Display current tasks.
  - Status indicators (New, In Progress).

## Flow
1. User navigates from Main Menu.
2. User taps Microphone to speak a task OR types it.
3. Task appears in the list.

## Visual Style
- **Standard Form**: Uses standard input fields and buttons.
- **List**: Card-based list for tasks.
